# Backend Robô django

## Run
    - (de preferência criar venv)
    Na raiz do projeto
        - pip install -r requirements.txt
        - fab dev
        - python3 manage.py runserver 
## DB
    Na raiz do projeto (sem uso)
        - sudo docker-compose up
        - docker ps 
## Swagger
    http://127.0.0.1:8000/api/swagger/
